### To Run the project
### Test the connection in MongoDB Atlas

1. Go to MongoDB Atlas    --- website

```bash
Click your cluster → Connect → Drivers.

Copy the “connection string for Node.js” it provides.

Replace the URI in your .env file with that string.

```
